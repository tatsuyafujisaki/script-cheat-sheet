# Shows installed and installable versions of Python, regardless of whether they were installed by UV.
```shell
uv python list
```

# Shows all installed Python versions, regardless of whether they were installed by uv.
```shell
uv python list --only-installed
```

# Runs Python 3.14
```shell
uv run --python 3.14
```

# Uninstalls all versions of Python installed by uv.
```shell
uv python uninstall --all
```
