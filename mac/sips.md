# What does `sips` stand for?
Scriptable Image Processing System

# How to show all the information of an image
```shell
sips -g all input.png
sips -g all input.jpg
sips -g all input.webp
```

# How to resize the width of an image
```shell
sips --resampleWidth 1024 input.png
sips --resampleWidth 1024 input.jpg
sips --resampleWidth 1024 input.webp
```
